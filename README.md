# carless-whisper-tracker
Setup

git clone https://github.com/USERNAME/careless-whisper-tracker.git
cd careless-whisper-tracker

Start Node:

npm init -y


Necessary modules:

npm install @whiskeysockets/baileys qrcode-terminal

Run the script:

node tracker.js
